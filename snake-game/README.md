# 🐍 贪吃蛇 Snake Game

一个简单又好看的贪吃蛇游戏。

## 玩法

- 使用方向键控制蛇的移动
- 手机端支持触摸滑动控制
- 吃到红色食物得分
- 撞墙或撞到自己游戏结束

## 技术栈

- 纯 HTML/CSS/JavaScript
- Canvas 绘制
- 响应式设计，支持移动端

## 在线预览

部署到 Vercel 后访问

## 本地运行

```bash
# 方式1: 直接打开
open index.html

# 方式2: Python HTTP 服务器
python3 -m http.server 8080
# 访问 http://localhost:8080
```

## 部署

```bash
# Vercel
vercel --prod
```
